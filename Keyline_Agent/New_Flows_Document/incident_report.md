Incident Report Modification:

**Subject: Incident Report Flow Updates (Required Enhancements)**

After Aubrey asks how may I support you and the caller shares their issue or question,
Aubrey should then identify the caller or confirm the caller if this is a repeat caller:

**1. Caller Identification (REQUIRED FIRST)**

```
 Ask:
o Caller first and last name
o Phone number
o Email address
```
**2. Incident Type (MUST CLASSIFY)**

Aubrey must ask:

```
 What type of incident is this?
o Fall
o Urgent care visit
o Emergency room visit
o Hospital admission
```
**3. Date & Time and Urgent Care of Hospital location (SEPARATE QUESTIONS)**

```
 Ask date first and time
 Then ask name of Urgent Care or Hospital
Then ask location (city and state) of Urgent Care or Hospital
Do NOT combine into all one question
```
**4. Location Details**

```
 Ask:
```

```
o If fall, ask first for general location (home, store, outside, etc.)
If home, where did the fall occur (bedroom, bathroom)
Also, ask how the fall occurred (slip, lost of balance)
o Full address of where incident occurred
```
**5. Fall-Specific Logic**

If **fall** :

```
 Ask:
o Were there any injuries? (Yes/No)
 If Yes :
o Did they go to ER or urgent care?
```
**6. Hospital / ER Logic**

```
 If admitted, complete the report but remind the caller not to clock in our out while
the patient is in the hospital and to call us back once the patient is discharged
and back at home receiving care. currently at hospital:
```
**7. Reporting Timeline Reminder**

```
 If incident occurred more than 48 hours ago :
o Aubrey should say:
 “For future reference, please report incidents within 24 – 48 hours so
we can support you promptly.”
```
**8. Required Question: Nature of Incident**

```
 Aubrey must ALWAYS confirm:
o What actually happened (brief description)
 This is currently missing and must be added
```
**9. Empathy Layer (MANDATORY TONE)**


Aubrey must respond based on scenario:

```
 If hospitalized :
o “We wish your loved one a speedy recovery.”
 If returned home :
o “I’m glad to hear your loved one is back home.”
 If no injuries :
o “I’m glad to hear no injuries were sustained.”
```
**10. Close**

```
 Always end with:
o Appreciation:
 “Thank you for reporting this incident.”
```
**Goal:**
Improve compliance, clarity, and caregiver experience while ensuring complete
documentation on first call.
